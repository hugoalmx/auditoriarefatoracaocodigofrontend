## 📄 Sobre o Repositório

Este repositório reúne os exercícios práticos desenvolvidos ao longo das aulas, com foco em:

- Escrita de **HTML semântico e estrutural**
- **Auditoria de acessibilidade** seguindo boas práticas de HTML5 e CSS3
- **Refatoração de código** Front-End para melhorar qualidade, legibilidade e conformidade com padrões web

## 📚 Conteúdo por Aula

### Aula 1 — HTML Estrutural e Semântico
Prática guiada com foco na construção de páginas usando elementos semânticos do HTML5, como `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, entre outros.

### Aula 2 — Auditoria e Refatoração de Código Front-End
Atividade prática de identificação de problemas em código HTML/CSS existente e aplicação de refatorações para melhorar a qualidade, semântica e manutenibilidade.

### Aula 3 — Checklist de Auditoria de Acessibilidade
Aplicação de um checklist de acessibilidade baseado nas diretrizes do **HTML5** e **CSS3**, verificando pontos como uso correto de atributos `alt`, labels em formulários, contraste de cores, navegação por teclado e estrutura de headings.
